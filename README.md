# TestGen AI - AI Test Case Generation Agent

## Problem Statement
Writing test cases manually takes time and developers may miss important edge cases.

## Solution
TestGen AI is an AI-powered tool that generates normal test cases, edge cases, invalid input cases and unit test code from pasted source code.

## Features
- AI-powered test case generation
- Supports JavaScript, Python, Java, C and C++
- Generates normal, edge and invalid test cases
- Generates unit test code
- Copy result option
- Download report option
- Clean responsive UI

## Tech Stack
- React
- Vite
- Gemini AI API
- CSS

## How to Run

```bash
npm install
npm run dev
```

Open the local URL shown in terminal.

## How to Use
1. Get Gemini API key from Google AI Studio
2. Paste API key in the app
3. Paste your code
4. Select programming language
5. Click Generate Test Cases

## Demo Script
Developers spend a lot of time writing test cases manually.  
Our project TestGen AI automates test case generation using Gemini AI.  
The user can paste any code, select the programming language and generate normal cases, edge cases, invalid test cases and unit test code instantly.  
This improves developer productivity, testing coverage and code reliability.
